**Problem Statement:**
Given an integer array `arr`, find the **maximum sum** of any **three consecutive elements** in the array.

**Example:**

```
Input:  arr = [1, 3, 2, 5, 1, 1, 2, 3, 4]
Output: 10
Explanation:
The subarray [2, 5, 3] gives the largest sum of 10.
```

**Constraints:**

- `3 <= arr.length <= 10^5`
- `-10^4 <= arr[i] <= 10^4`
